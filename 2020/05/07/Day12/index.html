Day12 - IO，文件字符流，文件字符流

Posted on 2020-05-07 In Java SE

IO

流的分类

文件流 FileInputStream / FileOutputStream 数据流的读写都是基于文件的操作

缓冲流 BufferedInputStream / BufferedOutputStream / BufferedReader / BufferedWriter 数据流的读写都是基于内存的操作

File类

File类只能操作文件本身，但是不能操作文件内容。

递归遍历文件实现输出一个文件夹下的所有文件和文件夹：

文件字节流

read方法有一个返回值，返回值是读取的数据的长度。如果读取到最后一个数据，还会往后读一个，也就意味着当read方法返回值是-1的时候，整个文件就读取完了

注意：文件字节流非常通用，可以用来操作字符的文档，还可以操作任何类型的其他类型文件（图片，压缩包等等），因为字节流直接使用二进制。

文件字符流

字符流只适合操作内容是字符的文件
